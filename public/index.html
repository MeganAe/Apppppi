Endpoints

GET /api/ai

Envoyer un prompt à un modèle IA et recevoir une réponse. Le paramètre model est optionnel.

ParamètreTypeStatusDescription
promptstringRequisLe message à envoyer
modelstringOptionnelModèle à utiliser (défaut: v3)
# Requête simple
GET /api/ai?prompt=Bonjour, comment vas-tu?

# Avec modèle spécifique
GET /api/ai?prompt=Explique l'IA&model=gemini

# Réponse
{
  "success": true,
  "model": "v3",
  "prompt": "Bonjour, comment vas-tu?",
  "reply": "Bonjour ! Je vais bien..."
}
GET /api/models

Retourne la liste de tous les modèles disponibles.

GET /api/models

{
  "success": true,
  "models": ["v3", "turbo", "gemini", "..."],
  "default": "v3"
}
